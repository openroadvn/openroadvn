Readme file for the Public & Private Files module for Drupal
------------------------------------------------------------

This module extend the standard private download feature of Drupal. It allows
defining specific folders to be private

Installation:
  Installation is like with all normal drupal modules:
  extract the 'public_and_private_files' folder from the archive to the
  modules directory from your website (typically sites/all/modules/contributed).

Dependencies:
  No dependencies defined

Conflicts/known issues:
  No conflicts or known issues yet

Configuration:
  The configuration page is at admin/settings/file-system,
  where you can configure the 'Public And Private Files" module.
